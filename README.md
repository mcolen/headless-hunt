# Headless Hunt

A lightweight Chrome extension that toggles the header on New York Times puzzle pages for a distraction-free solving experience.

## Installation

1.  Download or clone this repository.
2.  Navigate to `chrome://extensions/` in your browser.
3.  Enable **Developer mode** (top right).
4.  Click **Load unpacked** and select the extension folder.

## Usage

* **Toggle:** Click the extension icon or press `Alt+Shift+H`.
* **Note:** The extension only activates on `nytimes.com` puzzle pages.

## Privacy

Runs entirely locally. Does not request permissions to read your browsing history.
